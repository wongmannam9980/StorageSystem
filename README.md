Tracking system
Your tracking system is a web application that lets a developer store an item, enter a user’s email, generate a QR code, and send that QR code so the user can later retrieve the item through the system.
When the user scans the QR code, the system automatically accesses the correct bin, supports item retrieval, and then returns the item or bin flow back to the waiting area while keeping activity records.

Household storage system
Your household storage system is a web-based storage management interface for organizing items in bins at home, with functions for storing, retrieving, tracking quantity, and managing item records.
It also supports system initialization, activity logging, photo upload for stored items, and rules such as preventing negative quantity values, which makes it more like a practical home inventory control panel.

Main difference
The tracking system is more focused on delivery-style retrieval and user access, especially through QR-based collection flow.
The household storage system is more focused on inventory-style home organization, where the main goal is managing what is inside each bin and keeping the storage status clear.# StorageSystem
